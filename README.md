# TA-Helper
teaching assistant helper tool

학원 조교로 일했을 때, 업무 처리방식에 불편함을 느껴 구현한 웹 서비스.

메일 기반으로 학생, 수업정보 파일을 공유하는 협업 방식에 비효율을 느끼고, 
조교 간 공유 자동화 웹 서비스를 직접 기획하고 구현했습니다.
